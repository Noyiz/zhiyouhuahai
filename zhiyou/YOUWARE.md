# 智游花海 · 天姿牡丹园 AI导览

## 项目简介

这是一个「智游花海」H5沉浸式导览网页，专为北京太阳宫天姿牡丹园打造，支持AI互动识花功能。

## 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 7
- **样式**: Tailwind CSS 3 + 内联样式（适合移动H5精细控制）
- **字体**: Google Fonts Noto Serif SC / Noto Sans SC

## 设计规范

### 配色系统

| 名称 | 色值 | 用途 |
|------|------|------|
| 宣纸白 | `#F8F4ED` | 页面底色 |
| 魏紫 | `#7E2639` | 主色调（标题、强调） |
| 姚黄 | `#F7DA94` | 辅助色（高亮、徽标） |
| 豆绿 | `#9FB084` | 辅助色（副标、点缀） |
| 墨色 | `#2C1A1E` | 正文色 |

### 字体规范

- 标题/品名：`Noto Serif SC`（仿楷书风格）
- 正文：`Noto Sans SC`（清晰易读）
- 行高：`1.6`（正文舒适阅读间距）
- 圆角：`12px`（图片容器）

## 页面结构

1. **Hero Section** - 全屏牡丹园意境图 + 书法风格标题「智游花海」
2. **园内导览** - AI生成导览地图，支持双指缩放
3. **拍照识花** - 磨砂玻璃质感圆形按钮，链接识花小程序
4. **牡丹十绝** - 水平滑动卡片，展示10大名品牡丹
5. **页脚** - 园区信息

## 牡丹十绝品种

魏紫、姚黄、豆绿、赵粉、二乔、黑花魁、白雪塔、洛阳红、青龙卧墨池、御衣黄

## 开发命令

```bash
npm install    # 安装依赖
npm run dev    # 开发模式
npm run build  # 生产构建
```

## 特别说明

- 识花小程序链接：`https://shihua.example.com`（占位符，实际使用需替换）
